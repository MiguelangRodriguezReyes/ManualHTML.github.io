Ejemplos de listas

Listas HTML

Las listas HTML permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en listas.

Ejemplo

Una lista HTML desordenada

  • Café
  • Té
  • Leche

Una lista de HTML ordenada

  1. Café
  2. Té
  3. Leche

Lista HTML desordenada

Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la <li> etiqueta. Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada:

Ejemplo

  • Café
  • Té
  • Leche

Lista HTML ordenada

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>. Los elementos de la lista se marcarán con números de forma predeterminada:

Ejemplo

  1. Café
  2. Té
  3. Leche

Listas de descripción HTML

HTML también admite listas de descripción. Una lista de descripción es una lista de términos, con una descripción de cada término. La etiqueta <dl> define la lista de descripciones, la etiqueta <dt> define el término (nombre) y la etiqueta <dd> describe cada término:

Ejemplo

Café
- bebida caliente color negro
Leche
- bebida fria color color blanco
