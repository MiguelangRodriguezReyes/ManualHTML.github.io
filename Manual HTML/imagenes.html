Ejemplos de formatos

Comandos

El comando <img> sirve para insertar imágenes en una página WEB.

El comando <img src=""> sirve para insertar imágenes específicando la ruta en una página WEB.

El comando <img alt=""> sirve para especificar un texto alternativo de la imágen en una página WEB.

El comando <img width="" height=""> sirve para especificar el ancho y el alto de una imágen en una página WEB.
