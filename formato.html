Ejemplos de formatos

Comandos

El comando <i> sirve para poner itálica cualquier texto, ya sea párrafo o títulos, incluso palabras en específico de un párrafo.

El comando <b> sirve para poner en negritas una palabra en específico de cualquier texto, sea párrafo o un título.

El comando <strong> sirve para poner o mostrar un texto importante de un párrafo o título.

El comando <em> sirve para enfatizar un texto de un párrafo o título.

El comando <mark> sirve para poner o mostrar un texto marcado de un párrafo o título.

El comando <small> sirve para poner o mostrar un texto más pequeño de un párrafo o título.

El comando <del> sirve para poner o mostrar un texto eliminado de un párrafo o título.

El comando <ins> sirve para poner o mostrar un texto insertado de un párrafo o título.

El comando <sub> sirve para poner o mostrar un texto en subíndice de un título.

El comando <sup> sirve para poner o mostrar un texto en superíndice de un título.
