Ejemplos de Párrafos

Párrafos HTML

El elemento HTML define un párrafo. Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

Pantalla HTML

No puede estar seguro de cómo se mostrará HTML. Las pantallas grandes o pequeñas y las ventanas redimensionadas crearán resultados diferentes. Con HTML, no puede cambiar la visualización agregando espacios adicionales o líneas adicionales en su código HTML.

Reglas horizontales HTML

La etiqueta "hr" define una ruptura temática en una página HTML y, con mayor frecuencia, se muestra como una regla horizontal.

Saltos de línea HTML

El elemento HTML "br" define un salto de línea.
